# PM Prompts

A practical prompt library for Product Managers who are building AI products, not just writing docs about them. Every prompt is designed for real PM work: discovery, strategy, execution, stakeholder alignment, AI feature specs, and communication.

## How to use
1. Copy a prompt from the relevant folder.
2. Paste it into your AI tool of choice.
3. Fill in the input context fields.
4. Run it and iterate with your real constraints.

- [04-stakeholder](./04-stakeholder) — Executive updates, alignment maps, objection handling

- [08-career](./08-career) — Resume rewrites, case studies, interview practice, networking
